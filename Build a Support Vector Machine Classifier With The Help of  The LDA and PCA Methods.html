Build a Support Vector Machine Classifier With The Help of The LDA and PCA Methods¶

Context and objective:

Our work will be focused on a numerical dataset which describes wine quality grades that range from 3 to 8 corresponding to 6 distinct classes. The main goal here will be to build a support vector machine classifier to predict the wine grades. We'll be using the principal component analysis and linear discriminant analysis methods for dimensionality reduction and compare them to see which one is best suited for our classifier.

Note: This code was written on Jupyter Notebook.
Language: Python.
Packages: numpy, pandas, time, sklearn, matplotlib, seaborn.

Application of the linear discriminant analysis method

In order to make feature selection and lower the dimension of our dataset we need to define the linear discriminants. To do that, we have to:

  • Calculate the overall_means and the class means.
  • Calculate the within-class scatter matrix.
  • Calculate the between-class scatter matrix.
  • Calculate the eigenvalues and the eigenvectors for the LDA.

# calculate the within-class scatter matrix
S_w = np.zeros((11,11)) # define the within-class scatter matrix

# the below code calculates the covariance matrix for each separate quality grade
# and then sums those individual class covariance matrices up, resulting in a within-class scatter matrix
for gr, mv_row in zip(range(3,9), mean_vectors):
  S_i = np.zeros((11,11)) # covariance matrix for the i-th class
  for x in X_scaled[y == gr]:
    x, mv_row = x.reshape(11,1), mv_row.reshape(11,1)
    S_i += (x - mv_row).dot((x - mv_row).T)
  S_w += S_i 
 
print("The within-class scatter matrix:\n", S_w)

S_b = np.zeros((11,11)) # define the between-class scatter matrix

for i, mv in enumerate(mean_vectors):
  n = X_scaled[y == i+3,:].shape[0]
  mv, overall_means = mv.reshape(11,1), overall_means.reshape(11,1)
  S_b += n * (mv - overall_means).dot((mv - overall_means).T) # the between-class scatter matrix formula

print("The between-class scatter matrix is:\n", S_b)

# we need the eigenvalues in order to find the percentage of variance carried by each of our linear discriminants (the eigenvectors)
# to find these proportions, we need to divide each eigenvalue by the sum of all eigenvalues and multiply that by 100
# this will give us the percentage of variance explained by the eigenvector associated with this eigenvalue
eig_pairs = [] # create an empty list

# the code below will add each eigenvalue paired with its associated eigenvector to the empty list
# for the negative eigenvalues we need to multiply the eigenvalues and the eigenvectors by -1
for i in range(len(eigvals)):
  if eigvals[i] < 0:
    eig_pairs.append((-eigvals[i], -eigvects[i])) # if an eigenvalue is negative, we'll take the negative of their corresponding eigenvector
  else:
    eig_pairs.append((eigvals[i], eigvects[i])) # if the eigenvalue is positive, we'll take the original eigenvector

eig_pairs = sorted(eig_pairs, key = lambda k: k[0], reverse = True) # order the eigenpairs in descending order with respect to the eigenvalues

# print the eigenvalues paired with their respective eigenvectors in descending order
print('Evalues in descending order:\n')
for i in range(len(eig_pairs)):
  print('Evalue: ')
  print(eig_pairs[i][0].real)
  print('Evector:')
  print(eig_pairs[i][1].real)
  print('\n')

Analysis of LDA:
To find the significance of the newly constructed axes, we added all the eigenvalues and divided each of them separately by the overall sum to obtain the percentage of retained variance.
We must project our data onto the discriminants that make up more than 80% of the total variance. In our case, the two first linear discriminants make up around 95% of the total variance in the data. We now have our two new linear discriminants that correspond to the two highest eigenvalues instead of the original 11 features.
